Class Color

Model class representing a color used in the pattern.

A color represents a color used in the pattern. The color is made up of one or more strands of thread. Each strand can be a different color, allowing blended colors to be defined.

  • Throws

    Error if any invalid parameters are provided.

    Parameters

    • colorId: number

      An id for the color. Stitches will use to reference which color they are.

    • colorName: string

      A name for the overall color.

    • patternSymbol: string

      The ASCII character used to represent the color on the pattern visually.

    • strands: Strand[]

      An array of strand objects defining all the strands of thread that make up the color.

    • Optional totalFullStitches: number

      The count of full stitches that use this color in the pattern. This is not required.

    • Optional totalThreeQuarterStitches: number

      The count of three quarter stitches that use this color in the pattern. This is not required.

    • Optional totalHalfStitches: number

      The count of half stitches that use this color in the pattern. This is not required.

    • Optional totalQuarterStitches: number

      The count of quarter stitches that use this color in the pattern. This is not required.

    • Optional totalBackStitches: number

      The count of back stitches that use this color in the pattern. This is not required.

    • Optional totalLongStitches: number

      The count of long stitches that use this color in the pattern. This is not required.
